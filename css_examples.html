Ejemplos de CSS

Ejemplo 1: Cambiar el color de fondo

Este ejemplo muestra cómo cambiar el color de fondo de un elemento utilizando CSS:

        
          .my-element {
            background-color: yellow;
          }

Para aplicar el estilo a un elemento en tu HTML, agrega la clase "my-element" al elemento:

        
          <div class="my-element">
            Contenido del elemento
          </div>

Ejemplo 2: Cambiar el tamaño de la fuente

Este ejemplo muestra cómo cambiar el tamaño de la fuente de un elemento utilizando CSS:

        
          .my-element {
            font-size: 20px;
          }

Para aplicar el estilo a un elemento en tu HTML, agrega la clase "my-element" al elemento:

        
          <p class="my-element">
            Este es un párrafo con un tamaño de fuente de 20px.
          </p>

Ejemplo 3: Estilos de borde

Este ejemplo muestra cómo aplicar estilos de borde a un elemento utilizando CSS:

        
          .my-element {
            border: 1px solid #ccc;
            border-radius: 5px;
            padding: 10px;
          }

Para aplicar el estilo a un elemento en tu HTML, agrega la clase "my-element" al elemento:

        
          <div class="my-element">
            Contenido del elemento
          </div>
