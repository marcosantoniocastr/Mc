Ejemplos de HTML

Ejemplo 1: Encabezados

Este ejemplo muestra cómo usar los elementos de encabezado en HTML:

        
          <h1>Encabezado de nivel 1</h1>
          <h2>Encabezado de nivel 2</h2>
          <h3>Encabezado de nivel 3</h3>

Ejemplo 2: Párrafos

Este ejemplo muestra cómo crear párrafos en HTML:

        
          <p>Este es un párrafo de ejemplo.</p>
          <p>Este es otro párrafo.</p>

Ejemplo 3: Enlaces

Este ejemplo muestra cómo crear enlaces en HTML:

        
          <a href="https://www.ejemplo.com">Enlace de ejemplo</a>

Ejemplo 4: Listas

Este ejemplo muestra cómo crear listas en HTML:

        
          <ul>
            <li>Elemento 1</li>
            <li>Elemento 2</li>
            <li>Elemento 3</li>
          </ul>
