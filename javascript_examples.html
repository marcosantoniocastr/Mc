Ejemplos de JavaScript

Ejemplo 1: Saludo personalizado

Este ejemplo muestra cómo mostrar un saludo personalizado utilizando JavaScript:

        
          var nombre = prompt("Ingrese su nombre:");
          var saludo = "Hola, " + nombre + "!";
          alert(saludo);

Ejemplo 2: Suma de dos números

Este ejemplo muestra cómo sumar dos números ingresados por el usuario utilizando JavaScript:

        
          var num1 = parseInt(prompt("Ingrese el primer número:"));
          var num2 = parseInt(prompt("Ingrese el segundo número:"));
          var suma = num1 + num2;
          alert("La suma es: " + suma);

Ejemplo 3: Generar números aleatorios

Este ejemplo muestra cómo generar números aleatorios utilizando JavaScript:

        
          var numeroAleatorio = Math.floor(Math.random() * 100) + 1;
          alert("Número aleatorio: " + numeroAleatorio);
