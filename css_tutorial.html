Tutorial de CSS

Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para dar formato y diseñar páginas web.

Sintaxis básica

La sintaxis de CSS consiste en seleccionar elementos HTML y aplicar estilos utilizando propiedades y valores. Por ejemplo:

      
        p {
          color: blue;
          font-size: 16px;
        }

En este caso, todos los elementos <p> se mostrarán en color azul y con un tamaño de fuente de 16 píxeles.

Selector de clase

Los selectores de clase permiten aplicar estilos a un grupo de elementos específicos. Para definir una clase, utiliza el prefijo "." seguido del nombre de la clase. Por ejemplo:

      
        .destacado {
          background-color: yellow;
          font-weight: bold;
        }

En este caso, todos los elementos con la clase "destacado" tendrán un fondo amarillo y un peso de fuente en negrita.

Selector de ID

Los selectores de ID permiten aplicar estilos a un elemento único. Para definir un ID, utiliza el prefijo "#" seguido del nombre del ID. Por ejemplo:

      
        #logo {
          width: 200px;
          height: 100px;
        }

En este caso, el elemento con el ID "logo" tendrá un ancho de 200 píxeles y una altura de 100 píxeles.
