Tutorial de HTML

Introducción a HTML

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para estructurar el contenido de una página web.

Elementos básicos

Los elementos básicos de HTML se utilizan para estructurar el contenido de una página. Algunos ejemplos son:

      
        <h1>Título principal</h1>
        <p>Este es un párrafo de ejemplo.</p>
        <a href="https://www.ejemplo.com">Enlace de ejemplo</a>

En este caso, se muestra un encabezado de nivel 1 (`h1`), un párrafo (`p`) y un enlace (`a`).

Atributos

Los atributos se utilizan para proporcionar información adicional a los elementos. Por ejemplo:

      
        <a href="https://www.ejemplo.com" target="_blank">Abrir enlace en una nueva pestaña</a>
        <img src="imagen.jpg" alt="Descripción de la imagen">

En el primer ejemplo, el enlace se abrirá en una nueva pestaña del navegador gracias al atributo `target="_blank"`. En el segundo ejemplo, se muestra una etiqueta de imagen (`img`) con el atributo `src` que especifica la ruta de la imagen y el atributo `alt` que proporciona una descripción de la imagen.

Listas

Las listas se utilizan para presentar información en forma de elementos numerados o sin numerar. Por ejemplo:

      
        <ul>
          <li>Elemento 1</li>
          <li>Elemento 2</li>
          <li>Elemento 3</li>
        </ul>
        
        <ol>
          <li>Elemento 1</li>
          <li>Elemento 2</li>
          <li>Elemento 3</li>
        </ol>

En este caso, se muestra una lista no numerada (`ul`) con tres elementos (`li`) y una lista numerada (`ol`) con los mismos tres elementos.
