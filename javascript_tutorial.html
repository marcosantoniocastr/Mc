Tutorial de JavaScript

Introducción

JavaScript es un lenguaje de programación de alto nivel utilizado para agregar interactividad y dinamismo a las páginas web.

Sintaxis básica

La sintaxis de JavaScript es similar a otros lenguajes de programación. Aquí tienes un ejemplo básico:


    // Declaración de una variable
    var mensaje = "Hola, mundo!";
    
    // Imprimir el mensaje en la consola
    console.log(mensaje);

Manipulación del DOM

JavaScript se utiliza para interactuar con el Document Object Model (DOM) y modificar el contenido de una página web. Por ejemplo:


    // Obtener un elemento del DOM
    var elemento = document.getElementById("miElemento");
    
    // Cambiar el contenido del elemento
    elemento.innerHTML = "Nuevo contenido";

Eventos

JavaScript permite responder a eventos, como hacer clic en un botón o mover el ratón. Aquí tienes un ejemplo:


    // Obtener el botón del DOM
    var boton = document.getElementById("miBoton");
    
    // Agregar un evento al botón
    boton.addEventListener("click", function() {
      alert("¡Hiciste clic en el botón!");
    });
